

# YouLoader

Простая консольная утилита для скачивания видео с YouTube.

## Возможности

-  Двуязычный интерфейс (Русский / Английский)
-  Скачивание видео в качестве HD, Full HD, 2K, 4K
-  Автоматическая установка зависимостей

## Быстрый запуск

### Способ 1: Запуск в один клик (рекомендуется)
Просто откройте `start.bat` двойным щелчком — программа сама установит все зависимости и запустится.

### Способ 2: Ручной запуск
```bash
python main.py
```
- Важно!
- Папка pyarmor_runtime_000000 обязана находиться в одной директории с main.py.

- Требования
- Python 3.7 или выше

- Как пользоваться
- Запустите программу через start.bat или командой python main.py

- Выберите язык (EN / RU)

- Введите старт (или start для английского)

- Вставьте ссылку на видео с YouTube

- Выберите качество: HD, Full HD, 2K или 4K

- Дождитесь окончания загрузки

- Видеофайл сохранится в папке с программой

- - Если загрузка не удалась, попробуйте снова через несколько минут или используйте VPN
